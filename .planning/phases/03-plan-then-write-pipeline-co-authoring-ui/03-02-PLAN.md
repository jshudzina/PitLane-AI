Returns list of `{"type": type_str, "offset": match.start()}`. Regex pattern: `r'\[JOURNALIST:\s*(quote|context|causal)\]'`

`_build_outline_prompt(year: int, round_num: int, angle_name: str, angle_rationale: str, act_data: dict) -> str` — builds the non-streaming prompt. Include in the prompt:
- The angle name and rationale
- The five act labels from ACT_CONFIG (e.g., "Act 1: Qualifying / Grid", etc.)
- Key data from act_data (one sentence per act)
- Instruction: return a JSON array of 5 objects with keys: beat_number (1-5), beat_title (string), data_anchors (string), act_number (int 1-5)
- System instruction: the outline must contain at least one [JOURNALIST: quote], one [JOURNALIST: context], and one [JOURNALIST: causal] placeholder reference in the data_anchors for each beat

`_build_beat_prompt(beat_title: str, data_anchors: str, full_outline: list[OutlineBeat], act_data: dict) -> str` — builds the per-beat streaming prompt. Include:
- The full outline (all beat titles and data anchors, numbered) — per D-02
- The current beat's title and data anchors
- Act data context for the relevant act
- Instruction: write prose for this beat. MUST include exactly one [JOURNALIST: quote] placeholder, one [JOURNALIST: context] placeholder, and one [JOURNALIST: causal] placeholder. These must appear verbatim in the prose, not filled in.
- Prose target: 150-250 words

def generate_outline(
    self,
    article_id: str,
    year: int,
    round_num: int,
    angle_id: str,
    angle_name: str,
    angle_rationale: str,

Implementation:
1. Instantiate `FiveActMapper()` and call `mapper.fetch_act_data(year, round_num, act_number)` for acts 1-5; store results in `act_data: dict[int, dict]`
2. Build `prompt = _build_outline_prompt(year, round_num, angle_name, angle_rationale, act_data)`
3. Call sync Anthropic: `client = anthropic.Anthropic(); response = client.messages.create(model="claude-haiku-4-5", max_tokens=2048, messages=[{"role": "user", "content": prompt}])`
4. Parse JSON from `response.content[0].text` — extract list of beat dicts; validate/coerce with `[OutlineBeat(**b) for b in raw_beats]`
5. Persist via `BeatStore().save_outline_beats(article_id, [b.model_dump() for b in outline_beats])`
6. Transition article: `ArticleStore().transition_status(article_id, "outline_generated")`
7. Return list of OutlineBeat
8. Wrap in try/except: catch `anthropic.AuthenticationError` → log warning + re-raise; catch `json.JSONDecodeError` → log error + re-raise with message "Outline LLM response was not valid JSON"; catch general `Exception` → log exception + re-raise

```python
async def stream_beat(
    self,
    article_id: str,
    beat_number: int,
) -> AsyncIterator[str]:  # yields SSE-formatted strings
```
Implementation (async generator — no gate check here; gate is in route handler):
1. Load outline: `beat_store = BeatStore(); outline_beats = beat_store.get_outline_beats(article_id)` — convert to list of OutlineBeat via `[OutlineBeat(beat_number=r.beat_number, beat_title=r.beat_title, data_anchors=r.data_anchors or "", act_number=r.act_number) for r in outline_beats]`
2. Find current beat: `beat = next((b for b in outline_beats if b.beat_number == beat_number), None)`; if None: `yield f"event: error\ndata: {json.dumps({'beat_number': beat_number, 'message': 'Beat not found', 'retryable': False})}\n\n"` and return
3. Load act data: `mapper = FiveActMapper(); article = ArticleStore().get(article_id); act_data = mapper.fetch_act_data(article.race_year, article.race_round, beat.act_number or beat_number)`
4. `yield f"event: beat_start\ndata: {json.dumps({'beat_number': beat_number, 'beat_title': beat.beat_title, 'total_beats': len(outline_beats)})}\n\n"`
5. Build prompt: `prompt = _build_beat_prompt(beat.beat_title, beat.data_anchors, outline_beats, act_data)`
6. Stream: `async with _async_client.messages.stream(model="claude-sonnet-4-5-20250929", max_tokens=1024, messages=[{"role": "user", "content": prompt}]) as stream: async for text in stream.text_stream: full_prose_parts.append(text); yield f"event: token\ndata: {json.dumps({'beat_number': beat_number, 'token': text})}\n\n"`
7. After stream: `prose = "".join(full_prose_parts); placeholder_markers = _detect_placeholders(prose)`
8. Persist: `BeatStore().save_beat(article_id, beat_number, beat.beat_title, prose, placeholder_markers)`
9. `yield f"event: beat_done\ndata: {json.dumps({'beat_number': beat_number, 'prose': prose, 'placeholder_markers': placeholder_markers})}\n\n"`
10. Wrap step 6-9 in try/except: catch any Exception → `yield f"event: error\ndata: {json.dumps({'beat_number': beat_number, 'message': str(e), 'retryable': True})}\n\n"`

Note: `full_prose_parts: list[str] = []` declared before the async with block.
